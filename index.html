# display : GRID

페이지를 여러 주요 영역으로 나누거나, 크기와 위치 및 문서 계층 구조의 관점에서
HTML 요소간의 관계를 정의하는 데 특화돼있다.

세로 열과 가로 행을 기준으로 요소를 쉽게 정렬할 수 있다.

BOX =
{ display: grid; }
{ grid-template-columns : repeat(3, 1fr); } = 가로로 3등분 하겠다.
{ grid-auto-rows : minmax(200px, auto); } = 세로는 한칸에 최소 200px 최대는 auto로 정한다

BOX
A
B
C
C
1 2 3 4
1 2 3 4 5 6

3등분하였기 때문에 가로로는 1 번부터 4 번까지 번호가 매겨진다

세로로는 따로 box에서 지정하지 않았기 때문에 item의 속성에 따라 기준이 달라진다.

A =
{ grid-column : 1; } = 가로기준 1 지점에서 한 칸만큼의 넓이를 가진다.
{ grid-row : 1 / 4; } = 세로기준 1 지점부터 4 지점까지의 영역을 차지한다.

B =
{ grid-column : 2 / 4; } = 가로기준 2 지점부터 4 지점까지의 영역을 차지한다
{ grid-row : 2; } = 세로기준 2 지점에서 한 칸만큼을 차지하겠다.

C =
{ grid-column : 3 / 4; } = 가로기준 3 지점부터 4 지점까지의 영역을 차지한다
{ grid-row : 3 / 6; } = 세로기준 3 지점부터 6 지점까지의 영역을 차지하겠다.

D =
{ grid-column : 1 / 3; } = 가로기준 1 지점부터 3 지점까지의 영역을 차지한다
{ grid-row : 3 / 5; } = 세로기준 3 지점부터 5 지점까지의 영역을 차지하겠다.
